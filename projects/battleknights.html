Battleknights GO

Galway Game Jam 7

Descripton

Battleknights GO is a turn based fight game created with a group for Galway Game Jam 7.
The game is fully playable and pretty well polished, we learned from our mistakes from our previous game jam, creating a simpler game and implemented everything we planned to. However, we simplified it so much that the gameplay felt a bit boring.

The theme for the Jam was "Charge", which we implemented by having basic attacks charge a power meter for a super attack.
