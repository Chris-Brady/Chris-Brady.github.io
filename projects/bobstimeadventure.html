Bob's Time Adventure

Galway Game Jam 5

Descripton

Bob's Time Adventure is a platformer game created with a group for Galway Game Jam 5. The theme for this game jam was "Warp", and although we did not have enough time to finish it, but it is playable.

We never had time to implement a death mechanic, so if you fall off the platforms you'll fall indefinitely and get a closer look at how the game is constructed.
This was one of our first game Jams and we learned a few valuable lessons from it, such as the importance of getting essential and core features finished first.
